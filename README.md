## 📄 Descripción

Programa que administra el registro de pacientes, la asignación de turnos,
la clasificación por prioridad, el control de atención y las estadísticas
diarias de un centro de salud.

## 🚀 Cómo ejecutar el proyecto

1. Clonar este repositorio:

```bash
git clone https://github.com/efrainleonelcaceres-art/TPIGrupoD16.git
```

2. Acceder a la carpeta del proyecto:

```bash
cd TPIGrupoD16
```

3. Ejecutar el archivo principal:

```bash
python TPI-GRUPOD16.py
```

> **Requisito:** Tener instalado Python 3.14 o superior.
